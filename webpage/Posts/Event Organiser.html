You have an event you are organising and wish to create an event app to enhance you attendee's experience. Event Organiser is a tool for creating such an event app. You will need to compile the relative data you wish to display into CSV files. Collect any images and text to be displayed and make a note of map locations and web pages.

Outline

Download EventOrgainser from the downloads page. Install and run. You should see something like. The application is split into two parts plus a tab control at the top.
  • Left-hand side: This is where you add delete and control the contents of you event app.
  • Right-hand side: This gives a fully working version of the event app you are constructing looks like. You can use this model to navigate between areas of your event app.
  • Top level tab control: Navigate between different left-hand control panels.
    1. File tab: Various load and save options. Plus name your event app.
    2. Data tab: Enter and structure all the data your event app is going to use.
    3. Display tab: Configure how your event app will look.
