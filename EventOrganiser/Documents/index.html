Event Organiser

Event Organiser is a GUI for creating data for our Event App mobile application.

The output data is generated in the form of a JSON data file. This file can then be placed somewhere on the internet where it can be accessed by our Event App. We can consider Event Organiser to be specialised a JSON file editor.

The right-hand display is for user input. The left-hand display gives a picture of what an Easy Event App mobile app would look like after displaying the JSON file on which we are working.
