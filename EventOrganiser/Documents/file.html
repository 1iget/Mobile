File tab

Load and save the .json file you are editing to various locations. Also start a new event from this tab.



New event

Clears the current event information except the dispaly. Sets the event title to an empty string.

Download from url

Downloads an event from a url. This assumes the event is stored in the Event App JSON format. This has to a raw text file.

Save from file

Saves the Event App data in the form of a JSON file to a local file. This event can then be worked on at a later date, by loading te event back into Event Orgainser. The current or last saved event is also backed up in the computer memory and automatically loaded on restart.

Load from file

Loads a local file. This needs to a JSON file in the Event App format.
