Display tab

Configure the how the information in the Event App is displayed. Thre are five tabs

Common
- Load and Save display formats. Select from predefeind formats.
Drawer
- Configures display of drawers.
Menu
- Configures display of the two menues.
Toolbar
- Configues display of toolbar.
Strings
- Configue all the text desplayed on the Event App. Allows for translation of the event app into any languate.
